베스킨라빈스 클론 코딩

2023.03.22 (개인 프로젝트)

베스킨라빈스 홈페이지를 클론코딩한 웹페이지입니다. html, css, bootstrap를 사용하여 개발하였습니다.

html 태그, css 스타일, bootstrap 라이브러리의 사용법을 학습하기 위해 제작하게 되었습니다.

a 태그, select 태그 등을 사용하여 해당 이벤트를 사용해 보았으며 img 태그, i 태그를 사용하여 최대한 비슷하게 구성하였습니다.

그리드의 개념을 확실히 숙지하게 되었으며 원하는 위치에 원하는 모양으로 태그를 배치할수 있게 되었습니다.

첫 웹페이지를 개발해 보았습니다. 직접 만든 웹페이지가 신기하게 다가왔으며, 더 다양하고 실용적인 이벤트를 사용하여 실제 서비스 하는 웹페이지를 만들고 싶다는 목표가 생기게 되었습니다. 이후 개발하는 프로젝트에 실제로 사용할 만한 기능을 하나씩 넣게 되었는데 그 출발점이 되었다고 생각합니다.

컨셉, 아이디어
  • HTML, CSS, Bootstrap을 이용하여 베스킨 라빈스 홈페이지 클론코딩
  • a태그를 이용하여 contact us 페이지로 이동
  • 그리드 기능을 이용하여 화면 정렬
  • Bootstrap의 캐러셀 기능 이용하여 이미지 슬라이드 기능 구현
  • 이미지, 아이콘 등을 사용하여 최대한 비슷하게 꾸미기

TastyBlog

2023.05.24 ~ 2023.05.31 (개인 프로젝트)

각 지방을 관광지와 맛집을 소개, 취미로 독특한 카페를 찾아다니는 일상공유하기 위한 웹페이지입니다.

가까운곳에서 새로움을 찾고싶은 사람, 여행을 좋아하는사람, 맛있는 식당을 찾고싶은 사람, 커피를 좋아하는 사람들에게 관심사를 공유하고 싶어 만들게 되었습니다.

각종 블로그, 인스타그램, 싸이월드 방명록 등을 밴치마킹하였으며 여행/맛집, 카페 두가지 테마의 블로그로 구성하였습니다.

MySQL을 사용하여 게시물 CRUD구현하였으며 DB개념 및 쿼리문에 대해 알게되었습니다. Springboot를 사용하여 MVC2 패턴으로 프로젝트를 개발하였고 MVC1과 MVC2의 차이점과 MVC2의 유지보수 용이성에 대해 알게되었습니다.

컨셉, 아이디어
  • 각 지방을 관광지와 맛집을 소개, 취미로 독특한 카페를 찾아다니는 일상 공유
  • 여행, 맛집 테마(일반 블로그 형식), 카페 테마(인스타그램 형식) 포스팅
  • 싸이월드 방명록 밴치마킹
  • 페이징, 검색, 로그인 시 아이디 저장, 리모컨 기능 등 편의성 제공
  • 관리자 계정 구현

밥뭇나(BMN)

2023.06.28 ~ 2023.07.13 (4인 팀 프로젝트)

식사 메뉴를 고민하는 사람들을 위한 맛집 추천 웹사이트입니다.

넘쳐나는 허위 정보로 아까운 시간을 낭비할때 해당 웹 사이트를 이용하여 약간의 시간으로 양질의 정보를 제공받을 수 있었으면 하는 마음에 개발하게 되었습니다.

처음으로 진행하게 된 팀 프로젝트였고 기획단계부터 회의를 진행하여 차근차근 단계를 밟아 개발하게된 웹사이트입니다.

jQuery를 처음으로 실용성 있게 활용하였습니다. child, find등 요소 검색 명령어를 사용하여 원하는 시점에 원하는 태그에 이벤트를 줄 수 있었습니다. Javascript와 jQuery를 통해 동적인 페이지를 만들 수 있단 것을 실감하게 되었습니다.

ajax와 thymeleaf를 사용하여 예약페이지를 개발하였습니다. 비동기 통신으로 실시간 예약정보를 반영하여 화면에 랜더링 되도록 하였습니다. for문과 if문을 활용하여 중복되는 코드사용을 줄였으며 긴 코드를 짧게 줄이는 것이 효율성과 완성도를 높인다는것을 실감하게 되었습니다.

컨셉, 아이디어
  • 부산 맛집을 평점, 테마 별 등 으로 보여주며, 추천 및 리뷰 작성이 가능한 웹사이트
  • 회원 등급제를 통한 신뢰성 있는 리뷰 및 평점 시스템
  • 가게 별 예약 기능
  • 나이, 성별, 카테고리 별 필터 + 검색 기능
  • 지도, 스마트에디터, 도로명 주소 등 다양한 api 활용

Project Inside

2023.07.31 ~ 2023.08.29 (3인 팀 프로젝트)

개발자에게 중요한 역량인 알고리즘 지식을 쌓기 위한 문제 제공서비스와 팀 단위의 프로젝트를 준비하거나, 참여를 희망하는 개발자들을 위한 매칭 웹사이트입니다.

편리한 알고리즘 풀이와 연계되는 프로젝트 팀 형성을 통한 유저의 역량강화와 다양한 경험으로 성장기회를 제공하고 개발자들의 성장 발판이 되고자 개발하였습니다.

Node.JS, React 등 새로운 기능을 사용해 보았습니다. 유용한 모듈이 많았으며 수많은 모듈 중 적합한 모듈을 선정하는 과정이 필요했습니다.

크롤링을 사용해 보았습니다. 자동으로 원하는 페이지의 정보를 얻어오는 점은 유용하였지만 속도가 느려 실용적이지 못하다는 느낌을 받았습니다. 동일한 기능의 api도 사용했는데 속도가 빠르고 정확하였습니다. 필요한 정보가 특정되어 있는 상황에서는 api가 유용하고 실용적이라는 생각을 하게되었습니다.

react 사용 시 너무 많은 비동기가 이뤄지면 안된다고도 느꼈습니다. 로딩되는 순서에 따라서 정보를 못 얻어올 때가 있었기 때문입니다. aync와 await를 사용하여 pomise 객체가 나올때만 다음 통신이 이뤄지도록 순서를 정해주는 방식으로 해결하게 되었습니다. 계획적인 비동기 통신사용의 필요성을 실감하였습니다.

컨셉, 아이디어
  • 개발자들을 위한 자기개발 사이트
  • 본인 인증(이메일 인증)을 통한 로그인
  • 알고리즘 문제 공부, 실제로 코드가 동작하도록 하기
  • 토이 프로젝트 모집, 회원 별 레벨을 부여하여 원하는 수준의 팀원 모집하기
  • 채팅, 타이머, 간트 차트 등 다양한 편의성 제공
